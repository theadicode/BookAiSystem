# BookAiSystem

 📚 BookAI System

A simple AI-powered Book Management System built using Django.
Users can add books and get automatically generated summaries.


🚀 Features
	•	Add books (title + author)
	•	Store in database
	•	Auto-generate summaries
	•	Clean frontend UI


🛠️ Tech Stack

Python | Django | HTML

git clone https://github.com/theadicode/BookAiSystem
cd book-ai-system/backend
pip install django
python manage.py migrate
python manage.py runserver
